# CADRelay

Minimal hybrid CAD review MVP with future-ready architecture.

## Project structure
- docs/
- frontend/
- backend/
- worker/

## Stack (MVP)
- Frontend: React (to be scaffolded)
- Backend/API: FastAPI (Python)
- Worker: Python background service
- Metadata store: Firestore (planned)
- File storage: Object storage (planned)
